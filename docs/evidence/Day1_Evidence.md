## 完成任务

### Task1 项目背景理解

校园轻集市是一个校园二手闲置交易前端应用，解决校内闲置物品流转问题。核心场景包括商品浏览、发布、详情、搜索和用户系统。七天实训目标是完成一个可运行的 Vue 3 前端项目，并体验完整软件工程流程。

## 实验思考

1. **为什么第一天不立即开发页面？** — 先理解业务和架构，方向比速度重要。没有全局认知就开始编码，后期必然返工。
2. **"理解项目"为什么比"开始编码"更重要？** — 软件工程中需求理解和架构设计决定了项目成型后的质量，代码只是执行层。
3. **AI 应承担什么角色？** — 辅助者，不是替代者。能帮你解释代码、生成模板、检查错误，但架构决策和业务理解必须自己掌握。
4. **今天最大收获？** — 从零理解了 Vue 3 工程的完整骨架，不再觉得项目目录是"黑盒"。
